FoodForward SA Website
A simple website for FoodForward SA, a non-profit organization dedicated to fighting hunger in South Africa by redistributing surplus food.

Project Structure
FoodFoward SA.html - Homepage with mission statement and main image
About page.html - Information about the organization's vision and work
Events page.html - Upcoming events and social media links
Contact page.html - Contact form and email information
styles.css - Stylesheet for consistent design across pages
Features
Responsive design that works on mobile and desktop
Navigation menu on all pages
Contact form for inquiries
Events calendar
Social media integration
Mobile-friendly layout
Design Elements
Color scheme:
Primary green: #2d7c31
Background: #ffffff
Navigation: #e0e0e0
Consistent header and footer across pages
Flexbox layout for About page sections
Responsive images
Clean typography using Arial font family
Setup
Download all files
Keep files in the same directory
Open FoodFoward SA.html in a web browser to view the site
Browser Compatibility
The website is built with standard HTML5 and CSS3, compatible with modern web browsers.
